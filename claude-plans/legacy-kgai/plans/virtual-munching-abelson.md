**Tech stack (all from CDN, zero build step):**
- **Tailwind CSS** — styling
- **Chart.js** — bar/scatter/line charts
- **Vanilla JS** — interactivity (tabs, filters, year toggle)

#### Dashboard Layout

**Header Bar**
- Title: "GATE Cheating Detection Analysis"
- Subject toggle pills: `CS` | `EC` (switches all charts)
- Year toggle: `2025` | `2026` | `Compare` (overlay mode)
- Key stats: total attempts, centers, global mean, global σ

**Section 1: Suspicion Summary (Top of page)**
- Sortable table of centers with ≥2 flags
- Columns: Center, Year, Flag Count, Individual flags (colored chips)
- "NEW in 2026" badge for centers with 0 flags in 2025
- "WORSENED" badge with `0/7 → 4/7` style indicator
- Click a row → highlights that center across all charts below

**Section 2: Score Distribution (Analysis 1)**
- **Chart A**: Horizontal bar chart — top 30 centers sorted by mean score, global mean as vertical dashed line
- **Chart B**: Histogram — distribution of ALL center means (bin width 2 marks). Shows whether the overall distribution is normal or has outlier bumps
- YoY overlay: 2025 bars + 2026 bars side-by-side

**Section 3: Top Percentile Concentration (Analysis 2)**
- **Chart**: Scatter plot — X = center size (N), Y = top-1% concentration ratio
- Dot size = number of top-1% candidates
- Color: red if ratio > 5, orange if > 3, gray otherwise
- Hover tooltip: center name, N, top1%, ratio

**Section 4: Hard Question Performance (Analysis 3)**
- **Chart**: Bar chart — top 20 centers by hard-Q correct rate
- Horizontal dashed line at global hard-Q rate
- Color gradient: green (normal) → red (anomalous) based on Z-score
- Sidebar: list of the 10 hardest questions with their global correct rates

**Section 5: Wrong Answer Agreement (Analysis 4)**
- **Chart**: Horizontal bar chart — top 30 centers by agreement rate
- Vertical dashed line at 25% baseline (random chance)
- Color: red if >35%, orange if >30%, blue otherwise
- Labels show candidate count to contextualize

**Section 6: NAT Clustering (Analysis 5)**
- **Chart**: Bar chart — top 20 centers by suspicious NAT cluster count, **normalized by center size** (clusters per 100 candidates)
- Raw count shown in tooltip alongside normalized rate
- Fixes the false-positive issue with large centers like TCS Gito Bitan

**Section 7: Shift Leakage (Analysis 6)**
- **Chart**: Diverging bar chart — deviation from global shift delta
- Centers sorted by deviation, bars extend left (shift 1 advantage) or right (shift 2 advantage)
- Global Δ shown as annotation
- Only shown for multi-shift subjects (CS, CE)

**Section 8: Score Clustering (Analysis 7)**
- **Chart**: Bar chart — top 20 centers by cluster percentage
- Threshold lines at 35% (elevated) and 50% (high)
- Tooltip shows the specific score bucket (e.g., "32-34 marks")

**Section 9: Year-over-Year Comparison**
- **Chart**: Grouped bar chart — for each center appearing in both years, show flag count in 2025 vs 2026
- Only centers with ≥2 flags in either year
- Sorted by 2026 flag count descending
- Highlights "newly flagged" and "worsened" centers with badges

### Data JSON Structure

```typescript
interface DashboardData {
  metadata: {
    generatedAt: string;
    subjects: string[];
    years: number[];
    minCenterSize: number;
  };
  overview: {
    [year: number]: {
      totalAttempts: number;
      subjectBreakdown: { subject: string; count: number }[];
      distinctCenters: number;
    };
  };
  analyses: {
    [subjectGroup: string]: {
      [year: number]: {
        globalMean: number;
        globalStdDev: number;
        centerScores: CenterStats[];           // Analysis 1
        topPercentile: TopPercentileEntry[];    // Analysis 2
        hardQuestions: HardQuestionEntry[];     // Analysis 3
        hardQuestionList: { qId: string; type: string; correctRate: number }[];
        wrongAnswerAgreement: WrongAnswerEntry[]; // Analysis 4
        natClustering: NATClusterEntry[];       // Analysis 5
        natClusteringSummary: { center: string; count: number }[];
        shiftLeakage: ShiftLeakageEntry[];      // Analysis 6
        globalShiftDelta: number;
        scoreClustering: ScoreClusterEntry[];   // Analysis 7
      };
    };
  };
  suspicion: SuspicionEntry[];  // Composite summary
  yoyComparison: {
    newIn2026: SuspicionEntry[];
    worsened: { center: string; from: number; to: number; diff: number }[];
  };
}
```
